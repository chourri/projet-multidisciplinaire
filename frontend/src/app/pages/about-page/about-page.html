Climate Extreme Event Prediction System
Approche Neuro-Symbolique

Le Climate Extreme Event Prediction System (CEEPS) est un projet multidisciplinaire réalisé à l’ENSA Khouribga, dans le cadre de la filière Informatique et Ingénierie des Données.

Il vise à concevoir un système intelligent d’aide à la décision capable de prédire les événements climatiques extrêmes à partir de données météorologiques historiques.

Le projet repose sur une approche hybride neuro-symbolique, combinant le Deep Learning (LSTM), le Big Data, l’ingénierie des connaissances et une application web interactive pour la visualisation des résultats et des alertes climatiques.

Une étude de cas portant sur la ville de Marrakech (2010–2024) a permis de valider l’efficacité du système dans un contexte réel.
