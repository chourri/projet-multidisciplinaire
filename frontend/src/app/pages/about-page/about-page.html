Project Specification
CEEPS Initiative

The Climate Extreme Event Prediction System (CEEPS) is an engineering response to climate volatility in the Beni Mellal-Khenifra region. It bridges the gap between raw Big Data and actionable decision-making by fusing Deep Learning (LSTM) with Symbolic Logic.

Affiliation
ENSA Khouribga
University Sultan My Slimane

Scientific Objective

To develop a resilient forecasting architecture capable of detecting thermal and hydrological anomalies 7 days in advance. By prioritizing Neuro-Symbolic AI, we aim to eliminate the "Black Box" problem, ensuring that every alert issued is backed by transparent, traceable logic.

Technological Stack

  • Frontend: Angular 17 + Tailwind (Signals Architecture)
  • Backend: Node.js / Express + Python Bridge
  • AI Core: TensorFlow (LSTM) + Custom Rule Engine
